# Zellij Plugin

Terminal workspace manager: <https://zellij.dev/>

## Installation

```zsh
install-zellij
```

Requires [Homebrew](https://brew.sh/).
Utilities work regardless of installation method.

## Functions

- `zj`: Alias for `zellij`
- `za [session]`: Attach to session or create it (defaults to dir name)
- `zellij-config`: Edit zellij configuration directory
- `zellij-copy-layouts`: Copy bundled layouts to zellij config dir

## Environment

- `ZELLIJ_CONFIG_DIR`: `~/.config/zellij`
